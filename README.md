Coffee Shop Website 
This is a simple and responsive coffee shop website created using HTML , CSS and Javascript , It includes sections like Home , About , Menu and Contact to represent a small cafe or Coffee shop online.
Features :
  1. Simple and clean UI
  2. Menu section with items
  3. About section with details
  4. Contact Information
  5. Works on Mobile , tablet and laptop
Technologies used :
  * HTML
  * CSS
  * Javascript
